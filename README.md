# توضیحات مختصر
سورس کدی که توی این ریپازتوری قرار داره مربوط به یک کلاینت اپلیکیشن چت انلاین تحت وب میشه ، این برنامه از هیچ پایگاه داده ای استفاده نمیکنه بنابراین گفتگو ها ذخیره نمیشوند.

# ابزار های مورد استفاده 
کلاینت این برنامه از ریکت، تیل ویند و همچنین پکیج سوکت ای او کلاینت برای اتصال به سرور استفاده میکنه.

# نحوه استفاده
برای تبادل پیام در این برنامه کاربران میبایست در چت روم های مشابه قرار بگیرند بنابرین هنگام ورود شناسه گفتگو یکسان وارد کنید.

پس از ورود به گفتگو از طریقی اینپوتی که در پایین صفحه قرار گرفته پیامی را وارد کرده و سپس ارسال کنید، تمامی افراد حاضر در گفتگو پیغام شما را دریافت خواهند کرد. 
# نحوه اجرا 
ابتدا سرور را که با نام simple-chat-app-server  بر روی همین اکانت وجود دارد دریافت و با دستور npm start اجرا کنید.

سپس برای اجرا کلاینت در حالت دولوپمنت دستور زیر رو وارد کنید
npm run dev
